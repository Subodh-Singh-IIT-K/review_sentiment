# review_sentiment
## 🤖 Sentira-Ops: Deep Learning Sentiment EngineAn end-to-end NLP pipeline that transforms raw, noisy text into semantic sentiment predictions using a custom-trained Multi-Layer Perceptron (MLP). This project bridges the gap between classical text processing and modern deep learning optimization.
## 📌 Project OverviewThe goal of Sentira-Ops is to solve the "Semantic Gap" found in traditional text analysis. While basic models use One-Hot encoding (which lacks word relationships), this engine uses Contextual Embeddings to understand that "Excellent" and "Great" are mathematically similar.
## 🛠️ Technical Implementation
## 1. Data Engineering (Phase 1)Cleaning: Implemented a TextCleaner class using spaCy to handle HTML noise, punctuation, and case normalization.Lemmatization: Unlike simple stemming, I used Lemmatization to reduce words to their dictionary root (e.g., "running" → "run"). This significantly reduces Vector Sparsity, as noted in my foundational research.Stop Word Removal: Filtered high-frequency, low-info words (the, is, at) to focus the model's weights on sentiment-bearing tokens.
## 2. Neural Architecture (Phase 2)Built a 4-layer Neural Network using PyTorch:Input Layer: 384-dimensional dense vectors (Sentence Transformers).Hidden Layers: Fully connected layers with ReLU activation to prevent the "Vanishing Gradient" problem.Output Layer: 2-node linear layer with Softmax probability distribution for binary classification (Positive/Negative).
## 3. Optimization & Training (Phase 3)Loss Function: Utilized CrossEntropyLoss to measure the error ($E$) between predictions and ground truth.Momentum-based Learning: Employed the Adam Optimizer. By utilizing the Momentum term, the model successfully avoids "Zero Slope" traps (Local Minima), ensuring the gradient descent rolls toward the global minimum of the loss landscape.Backpropagation: Automated weight updates using the Delta Rule logic through PyTorch’s autograd engine.
## 🚀 Key FeaturesHigh-Dimensional Embeddings: Replaces sparse One-Hot vectors with dense semantic embeddings.Interactive Inference: Includes a real-time CLI loop for user-input testing.Confidence Scoring: Provides not just a label, but a probability score for every prediction.📈 PerformanceDataset: 5,000 IMDB Movie Reviews.Optimization: Trained over 10 epochs with a decreasing loss curve, demonstrating effective weight convergence.
## 💻 How to RunOpen the .ipynb file in Google Colab.Install dependencies: pip install spacy sentence-transformers torch.Run all cells to train the model.Use the start_sentiment_bot() function at the bottom to test your own reviews!
